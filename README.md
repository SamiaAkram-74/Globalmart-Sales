# Globalmart-Sales
1. Load & Inspect Data: Import the dataset. Check for missing
values, data types, and outliers.
2. Clean the Data: Handle missing values, duplicates, and inconsistent formatting.
3. Analyze & Visualize:
   . Identify top-selling products, cities, and customer demographics.
   . Create visualizations (histograms, box plots, heatmaps) using Matplotlib and Seaborn.
4. Preprocess: create time features (month, day, promo flags), handle missing values.
5. Train models: Linear Regression, Random Forest, and XGBoost (or LightGBM).
6. Evaluate: RMSE, MAE, and R2. Use time-based cross-validation
7. Explain: plot feature importance and residual analysis.
